Hotel Booking Application
---------------------------------------------
This application performs the following features:

1- Add/Delete/Update/Find/View all Customer/s
2- Add/Delete/Update/Find/View all Hotel/s
3- Add/Delete/Update/Find/View all Booking/s
4- Possibility that one customer can make multiple bookings
5- should recognize new customers from existing ones
6- The application should have separate menus for Different users
7- Saves all created client/hotel/Booking to Database
8- applies changes to database
